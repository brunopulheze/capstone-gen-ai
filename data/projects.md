# NexaLab — Projects

## Orbitly — AI-Powered Productivity & Goal Tracking Platform

**Live site:** https://orbitly.app
**Type:** Full-stack SaaS web application
**Role:** Sole designer and developer — responsible for every layer from brand identity to deployment

### Overview
Orbitly is a full-featured, AI-powered productivity platform that helps individuals and small teams set, track, and achieve goals. Users get personalised weekly plans, smart progress nudges, retrospective summaries, and an AI coach that adapts recommendations based on past behaviour.

The platform is fully internationalised (English, Spanish, French, German), GDPR-compliant, and monetised through Stripe subscriptions and one-off credit purchases.

### Tech Stack
- **Frontend:** Next.js 15 (App Router), React 19, TypeScript, Tailwind CSS 4, Framer Motion
- **Backend:** PostgreSQL with Prisma ORM, NextAuth authentication
- **AI:** Groq's Llama 3.3 models for coaching and plan generation
- **Payments:** Stripe (subscriptions, one-off purchases)
- **Other:** Resend (email), Cloudflare Turnstile (bot protection), jsPDF (PDF export)
- **Deployment:** Vercel (serverless)

### Key Features
- AI-generated weekly goal plans personalised per user
- Daily check-ins with sentiment analysis and streak tracking
- Smart retrospectives — AI-written summaries at the end of each week
- Team workspace with shared goal boards
- Multi-language support (EN, ES, FR, DE)
- Full Stripe payment integration with subscription tiers

### Design
- Deep teal (#0d5c63) paired with warm amber (#f5a623) — evoking focus, energy, and forward momentum
- Clean card-based layout with subtle micro-animations on progress milestones
- Typography: Sora (headings), Inter (UI), Lora (long-form content)

## GreenRoots — Brand Identity & Web Application

**Live site:** https://www.greenrootsmarket.com
**Type:** Brand identity + full-stack web app MVP
**Role:** Designer and developer
**Location:** Utrecht, Netherlands

### Overview
GreenRoots is a local organic food subscription service in Utrecht focused on weekly vegetable box deliveries from nearby farms. The project involved creating a complete brand identity from scratch and building an MVP web application.

### Tech Stack
- **Frontend:** React (functional components, hooks, ES6+), Atomic Design methodology
- **UI Library:** Chakra UI for consistent, accessible components
- **Backend:** Node.js, Express.js, MongoDB (MERN stack)

### Key Features
- Subscription box configurator with dietary preference filters
- Farm profiles and seasonal availability pages
- Order management dashboard for subscribers
- Responsive design across all screen sizes

### Design
- Primary colour: earthy green shades (#3a7d44, #1b4332) — reflecting nature, freshness and sustainability
- Accent colour: warm terracotta — grounded complementary hue conveying craft and locality
- Rounded, organic typography conveying friendliness and approachability
- Full style tile, typography system, colour palette documentation, and design system
